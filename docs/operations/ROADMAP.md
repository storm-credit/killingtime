# Execution Roadmap

## Phase 1

- establish fixed orchestra harness
- create config-first registry for agents, skills, hooks, and MCP
- launch a dashboard that reflects the registry

## Phase 2

- attach downloader, ASR, OCR, translation, and QA execution scripts
- connect actual storage for manifests and outputs
- add project-level edit screens

## Phase 3

- automate handoff between stages
- add terminology memory
- support multi-project queueing
- add release packaging per target platform
- only add video rendering if subtitle replacement requires it later
